CHARMANDER

Une application de bureau développée pour simuler une combustion dans n'importe quelle condition pour un réacteur partiellement remué par la méthode de la fonction de densité de probabilité.

Tâches en cours:

  • Appliquer l'architecture MVC (Model-View-Controller);
  • Changez l'interface graphique pour Kivy;
  • Documentation du logiciel;

Tâches futures:

  • Tester les modèles de mélange des autres;
  • Validation finale;
  • Publication.
  • Python était la principale langue utilisée dans ce projet;
  • Programmation orientée objet était l'une des principales caractéristiques utilisées;
  • Tkinter a été utilisé pour l'interface graphique;
  • Github a été utilisé pour le contrôle de version;
  • Cantera API pour les approximations thermodynamiques.
