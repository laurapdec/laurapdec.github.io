CHARMANDER

Uma aplicação desktop desenvolvida para simular uma combustão em qualquer condição para um Reator Parcialmente Agitado através do método de função de densidade de probabilidade.

Tarefas em andamento:

  • Aplicar a arquitetura MVC (Model-View-Controller);
  • Mudar a GUI para Kivy;
  • Documentação do software;

Tarefas futuras:

  • Testar outros modelos de mistura;
  • Validação final;
  • Publicação.
  • Python foi a principal linguagem utilizada neste projeto;
  • Programação orientada a objetos foi uma das principais características utilizadas;
  • Tkinter foi utilizado para a GUI;
  • Github foi usado para controle de versão;
  • Cantera API para aproximações termodinâmicas.
